# 💻 YtYt – Programming School Landing Page

A modern and responsive landing page for an online programming school. The website presents the school's mission, learning process, available technologies, course pricing, and student support in a clean and user-friendly interface.

## 🚀 Features

* 📱 Fully responsive design
* 🎨 Modern and clean UI
* 🏠 Hero section with call-to-action
* 💡 Technologies and programming stack overview
* 📚 Learning process explanation
* ⭐ Benefits and advantages cards
* 🎥 Video presentation section
* 👨‍🏫 Student support and learning methodology
* 💰 Course pricing table
* 📞 Footer with contact information and social media links

## 🛠️ Built With

* **HTML5**
* **CSS3**
* **Flexbox**
* **Responsive Web Design**
* **Google Fonts**

## 🎯 Project Goal

The purpose of this project is to practice front-end web development by recreating a modern landing page from a design mockup. The project focuses on semantic HTML, CSS layouts, responsive design techniques, and creating visually appealing user interfaces.
